# Digital Clock Dashboard

A web-based digital clock built using HTML, CSS, and JavaScript.

## Features
- Real-time digital clock
- Greeting based on time of day
- Weather information using API
- Dark mode toggle
- 12-hour / 24-hour format switch
- Dynamic background based on time

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## Purpose
This project was built to practice JavaScript concepts such as DOM manipulation, time functions, API usage, and UI interaction.
